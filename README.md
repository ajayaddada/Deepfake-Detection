Deepfake-Detection
=========================================

**Description**
------------
This Python-powered Deepfake Detection project uses AI models to classify uploaded images or videos as real or fake, all through an intuitive web browser interface.
Leverages Hugging Face Transformers for detection, Gradio for instant UI, and is optimized for fast, reliable demo and experimentation.

Perfect for media verification, social media safety, evidence validation, and hands-on machine learning research!
